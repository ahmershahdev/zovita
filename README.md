# Zovita

Premium healthcare commerce experience for medicines, supplements, and wellness essentials.

## Highlights

- Curated product catalog with category filters
- Premium product cards, KPIs, and insights
- Account dashboard experience
- Contact and support flows

## Tech Stack

- PHP (Laravel-ready structure)
- Tailwind CSS
- Vanilla JavaScript
- Vite (assets build)

## Getting Started

1. Install dependencies:
   - `composer install`
   - `npm install`
2. Copy environment:
   - `cp .env.example .env`
3. Run the dev server:
   - `npm run dev`
